#  CBJR Interactive Player

Projeto desenvolvido para praticar **HTML, CSS, JavaScript, Git e GitHub** através da criação de um player inspirado na discografia do Charlie Brown Jr.

# Status do projeto

Em desenvolvimento

Atualmente o projeto já possui uma interface funcional e estilizada utilizando Glassmorphism.

## ✅ Funcionalidades implementadas

* [x] Exibir capa do álbum
* [x] Exibir nome do álbum
* [x] Exibir ano de lançamento
* [x] Estruturação dos dados utilizando Arrays e Objetos
* [x] Atualização dinâmica da interface com JavaScript
* [x] Card com efeito Glassmorphism
* [x] Efeito Hover no card
* [x] Organização do código através da função `atualizarPlayer()`
* [x] Versionamento utilizando Git
* [x] Repositório hospedado no GitHub

## 🔜 Próximas funcionalidades

* [ ] Álbum anterior
